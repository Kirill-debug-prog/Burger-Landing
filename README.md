Одностраничный сайт для практики верстки

Репозиторий содержит одностраничный сайт, созданный с целью прокачки навыков работы с разметкой и стилями. Основные моменты:

- Практика HTML и CSS
- Создание адаптивного дизайна под различные устройства
- Использование современных подходов к верстке
- Применение hover-эффектов и базовой анимации

Цель проекта — улучшить навыки верстки и адаптивного дизайна.
